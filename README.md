# 👕 오늘 뭐 입지?
오늘의 기온과 풍속/습도를 통해 체감온도를 예측한 뒤 그에 맞는 옷을 추천해주는 웹 앱   

프로젝트 기간 : 23.06.13 ~ 23.06.16 (4일)    
프로젝트 인원 : 1명

## 📚 개발 환경
- Python
- Visual Studio Code
- Google Colab
- Flask
- Postgresql
- Pandas
- Numpy
- Seaborn
- Scikit-learn
- Joblib
- Psycopg2
